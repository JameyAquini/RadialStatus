# RadialStatus
A Discord theme that changes the default Discord status into a radial indicator around your profile picture. Minimal, clean, and easy to use.

# Step-by-step installation:
1. Download the .css theme file
2. Open Discord and go to Vencord Settings → Themes → Open Theme Folder.
3. Move the downloaded .css file into the theme folder.
4. Back in Discord, toggle the theme ON to activate it.

> [!IMPORTANT]
Make sure you have Vencord installed: https://vencord.dev/ (official link)
